## What is MCP Hub?

MCP Hub is an open-source registry for Model Context Protocol (MCP) servers. It provides a beautiful, searchable interface to discover, configure, and manage MCP servers for Claude Code and other MCP-compatible clients.

## Features

- **Browse & Search**: Find MCP servers by name, category, or tags
- **One-Click Install**: Copy Claude configuration with a single click
- **Custom Servers**: Add and manage your own MCP servers
- **Import/Export**: Share server configurations easily
- **Dark/Light Mode**: Comfortable viewing in any environment
